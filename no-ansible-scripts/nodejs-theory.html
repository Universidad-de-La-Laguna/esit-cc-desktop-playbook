El entorno de ejecución Node.js y APIs REST

Este repositorio pretende ir almacenando los diferentes aspectos a desarrollar en las clases de teoría de la asignatura en lo que respecta al entorno de ejecución Node.js y el desarrollo de un API REST (Representational State Transfer).

A continuación, puede encontrar un índice de los contenidos:

  1. ¿Qué es Node.js?
  2. El sistema de ficheros y creación de procesos
  3. Sockets
  4. Peticiones HTTP y patrones callback y callback-chaining
  5. Servidores Web a través de Express
  6. Promesas
  7. Operaciones CRUD con MongoDB
  8. Modelado de objetos con Mongoose
  9. API REST
  10. Mejorando el uso de las promesas con async/await
  11. Refactorizando el API REST mediante async/await
  12. Pruebas sobre el API REST
  13. Despliegue del API REST
